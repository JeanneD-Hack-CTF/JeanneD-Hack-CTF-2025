## Description

Dans un futur dystopique où la technologie et la cybernétique dominent, vous incarnez un cyber-hacker déterminé à percer les mystères du cyber-espace. Les corporations cachent des secrets inestimables, et chaque bit de données peut être la clé d'un pouvoir immense.

L'objectif de ce challenge est de découvrir la graine initiale d'un générateur aléatoire. En mettant à profit vos compétences en hacking, vous devrez analyser le code et traverser des systèmes complexes afin de récupérer le flag, qui représente votre triomphe sur les puissances obscures manipulant l'information.
